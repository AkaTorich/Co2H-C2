Co2H — руководство оператора

Co2H — C2-фреймворк (аналог Cobalt Strike) для red team операций. Доступен в двух версиях: Windows и Linux — teamserver и клиент работают на обеих платформах, beacon'ы генерируются под целевую ОС. Состоит из трёх компонентов:

КомпонентОписание
teamserverСерверная часть: HTTPS/TCP/SMB/DNS-listener'ы, БД SQLite, mTLS для операторов. Работает на Windows и Linux.
client (Qt6)Графическая консоль оператора: Sessions / Listeners / Graph / Downloads / Credentials / Chat / Logs + REPL. Работает на Windows и Linux.
beaconПолезная нагрузка: Windows (x64/x86 EXE/DLL), Linux (x64/ARM64/ARM32 ELF/SO), macOS (Universal/ARM64/x64 Mach-O/DYLIB).

Содержание

  • Быстрый старт
  • Интерфейс оператора (окно, меню, панель, вкладки)
  • Консоль beacon'а и подсказка команд
  • Справочник команд
  • Teamserver — конфигурация и запуск
  • Listener'ы (HTTPS, TCP, SMB, DNS, Pivot, SOCKS5, Reverse Port Forward)
  • Malleable-профили
  • Генератор артефактов
  • Sleep Mask Kit
  • Process Inject Kit
  • Plugin API — разработка плагинов
  • Диагностика и типовые проблемы
