Консоль beacon'а

Нижняя часть главного окна — REPL, отправляющий команды активному beacon'у по очереди задач. Вывод приходит асинхронно при следующем check-in'е.

Заголовок

No beacon selectedBeacon не выбран.
beacon <id>Активный beacon, обычный режим.
beacon <id> [interactive shell]Интерактивный cmd.exe на цели.

Подсказка команд

При наборе текста над полем ввода всплывает popup:

  • Префиксный ввод (например exec) → список совпадений (до 8): имя + аргументы + краткое описание.
  • Имя + пробел (например spawnto ) → детальная подсказка по этой команде.

Подсказка скрывается на пустом вводе, после Enter и в интерактивном shell.

Интерактивный shell

shell без аргументов запускает живой cmd.exe на цели. Каждая строка отправляется на STDIN cmd.exe, вывод печатается в консоль. Команда exit внутри интерактивного режима закрывает cmd.exe, не убивая beacon.

Пути с пробелами

Везде, где принимается несколько аргументов, пути с пробелами заключаются в двойные кавычки:

upload "C:\loot\report doc.docx" "C:\Windows\Temp\r.docx"

История команд

Все введённые команды доступны в текущей сессии клиента стрелками ↑ / ↓ и не сохраняются между запусками.

Вкладки консоли

Каждый beacon получает собственную вкладку в нижней панели консоли. При двойном клике по сессии (или через Interact) открывается вкладка для этого beacon'а. Переключение между beacon'ами — клик по заголовку вкладки.

  • Переименование — двойной клик по заголовку вкладки открывает поле ввода для задания произвольного имени. Пустое значение возвращает имя по умолчанию (user@host (id)).
  • Буфер — каждая вкладка хранит до 25 000 строк. При превышении старые строки отбрасываются. Прокрутка — стандартное колесо мыши.

ANSI-раскраска

Консоль распознаёт ANSI escape-последовательности (ESC[...m) в выводе beacon'а и отображает цветной текст. Это позволяет видеть раскрашенный вывод таких инструментов как linpeas, pspy, nmap --color и подобных — без потери цветовой разметки.

Поддерживаются:

  • Стандартные 16 цветов (SGR 30–37, 40–47, 90–97, 100–107)
  • Расширенная палитра 256 цветов (SGR 38;5;N / 48;5;N)
  • Атрибуты: жирный (ESC[1m), курсив (ESC[3m), подчёркивание (ESC[4m), сброс (ESC[0m)
