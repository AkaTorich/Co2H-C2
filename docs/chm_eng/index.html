Co2H — Operator Manual

Co2H is a C2 framework (similar to Cobalt Strike) for red team operations. Available in two versions: Windows and Linux — the teamserver and client run on both platforms, beacons are generated for the target OS. Consists of three components:

ComponentDescription
teamserverServer side: HTTPS/TCP/SMB/DNS listeners, SQLite database, mTLS for operators. Runs on Windows and Linux.
client (Qt6)Operator graphical console: Sessions / Listeners / Graph / Downloads / Credentials / Chat / Logs + REPL. Runs on Windows and Linux.
beaconPayload: Windows (x64/x86 EXE/DLL), Linux (x64/ARM64/ARM32 ELF/SO), macOS (Universal/ARM64/x64 Mach-O/DYLIB).
