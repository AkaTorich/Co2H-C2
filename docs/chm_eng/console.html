Beacon Console

The lower portion of the main window is a REPL that sends commands to the active beacon via the task queue. Output arrives asynchronously on the next check-in.

Header

No beacon selectedNo beacon selected.
beacon <id>Active beacon, normal mode.
beacon <id> [interactive shell]Interactive cmd.exe on the target.

Command Hints

While typing, a popup appears above the input field:

  • Prefix input (e.g. exec) → list of matches (up to 8): name + arguments + brief description.
  • Name + space (e.g. spawnto ) → detailed hint for that command.

The hint hides on empty input, after Enter, and in interactive shell mode.

Interactive Shell

shell without arguments starts a live cmd.exe on the target. Each line is sent to the STDIN of cmd.exe, output is printed to the console. The exit command inside interactive mode closes cmd.exe without killing the beacon.

Paths with Spaces

Everywhere multiple arguments are accepted, paths with spaces are enclosed in double quotes:

upload "C:\loot\report doc.docx" "C:\Windows\Temp\r.docx"

Command History

All entered commands are available in the current client session via the ↑ / ↓ arrow keys and are not saved between restarts.

Console Tabs

Each beacon gets its own tab in the bottom console panel. Double-clicking a session (or via Interact) opens a tab for that beacon. Switching between beacons — click the tab header.

  • Rename — double-clicking the tab header opens an input field for setting a custom name. An empty value restores the default name (user@host (id)).
  • Buffer — each tab stores up to 25,000 lines. When exceeded, old lines are discarded. Scrolling — standard mouse wheel.

ANSI Coloring

The console recognizes ANSI escape sequences (ESC[...m) in beacon output and displays colored text. This allows you to see colored output from tools such as linpeas, pspy, nmap --color and similar — without losing color markup.

Supported:

  • Standard 16 colors (SGR 30–37, 40–47, 90–97, 100–107)
  • Extended 256-color palette (SGR 38;5;N / 48;5;N)
  • Attributes: bold (ESC[1m), italic (ESC[3m), underline (ESC[4m), reset (ESC[0m)
